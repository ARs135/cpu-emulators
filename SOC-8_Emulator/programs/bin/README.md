# The bin folder
This is not a recycling bin

---

This folder is a place to store the assembled assembly code. To go find the source of the bin file, go to the src folder and find the file with the same name as the bin file 

*example: `fib.hex` go find `fib.asm` in `src/`*

Yes I know I used .hex even tho this is called the BINary folder, who cares? Don't be pedantic.
